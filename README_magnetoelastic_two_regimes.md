# magnetoelastic_breather_two_regimes.py

Скрипт делает три шага подряд:

1. считает атомный бризер в цепочке FPU-beta;
2. строит автономный магнитный профиль `s_n` в rotating frame;
3. запускает полную магнитную динамику с магнитоупругой накачкой от атомного бризера.

## Какие режимы уже заведены

### 1. `bottom_dark`
Параметры взяты в логике статьи для нижнего темного бризера:
- `D/2J = 0.16`
- `B = 0.15`
- `Omega = -0.28`
- `beta = 1.5`

### 2. `bottom_bright`
Параметры взяты в логике статьи для нижнего светлого бризера:
- `D/2J = 0.16`
- `B = -1.0`
- `Omega = 1.95`
- `beta = 0.40` по умолчанию как рабочее поле для динамики

## Что внутри важно

- Стационарный магнитный seed теперь берется не из случайного состояния, а из решения автономной задачи на профиль `s_n`.
- Для полной динамики используется нормировка на `2J`.
- Магнитоупругая связь вводится через три канала:
  - `j_n(t)`
  - `d_n(t)`
  - `b_n(t)`
- На краях добавлено усиливающееся затухание, чтобы гасить улетающие спиновые волны.
- Накачка включается плавно, а не ступенькой.

## Как быстро менять физику

Если нужно оставить только старый канал через анизотропию, в начале файла поставь:

```python
ENABLE_J_DRIVE = False
ENABLE_D_DRIVE = False
ENABLE_B_DRIVE = True
```

Если нужно сканировать согласование времен, меняй:

```python
RHO_TIME_SCALING = 0.02
```

Если нужно усиливать/ослаблять каналы связи, меняй:

```python
LAMBDA_J = 0.04
LAMBDA_D = 0.02
LAMBDA_B = 0.15
```
